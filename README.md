## Setup

Install dependencies.

```shell
npm install hexo-cli -g
brew install pandoc
npm install
```

## Run Run Run

```shell
hexo cl     # clean the cache files
hexo g      # generate static files
hexo s      # server start
hexo d      # deploy to github page
```
